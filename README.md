# Copy Code README

Copy code with template literals for easy paste to editors' whats supports markdown.

## Features

_Transform this:_
const a;

_to this:_
```
const a;
```

_with adding template literals and new lines:_
\`\`\`\n${copiedContent}\n\`\`\`

### Default keybinding

- Windows: ```Ctrl+Alt+C```
- Mac: ```cmd+option+C```
