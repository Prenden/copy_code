# Copy Code

Copy code with automatic code-block formatter with syntax highlight.

![Copy Code icon](images/copy-code.png)

## Features

_Transform this:_
const a;

_to this:_
``` js
const a;
```

_with adding template literals and new lines:_
\`\`\`\n${copiedContent}\n\`\`\`

### Default keybinding

- Windows: ```Ctrl+Alt+C```
- Mac: ```Cmd+Shift+F9```

**Enjoy!**
